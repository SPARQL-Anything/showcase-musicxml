# Showcase MusicXML

## Get list of XML files

```
java -jar sparql-anything-0.8.0-SNAPSHOT.jar -q queries/getXMLPaths.sparql
```

## Get list of CSV files

```
java -jar sparql-anything-0.8.0-SNAPSHOT.jar -q queries/getCSVPaths.sparql
```
